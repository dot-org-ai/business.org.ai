# Business

A business entity such as a company or organization that offers products or services to customers.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | The name of the business. |
| description | string | A brief description of the business. |
| website | URL | The official website of the business. |
| industry | string | The industry sector the business operates in. |
| foundedDate | date | The date the business was founded. |
| employees | integer | The number of employees in the business. |
| headquarters | string | The location of the business's headquarters. |
| revenue | number | The annual revenue of the business. |
| logo | URL | The logo image of the business. |

## Relationships

| Relationship | Target Type | Description |

| hasService | Service | Services provided by the business. |
| hasBrand | Brand | Brands owned by the business. |
| hasSubsidiary | Business | Subsidiary companies owned by the business. |
| hasPartner | Business | Partner businesses associated with the business. |
